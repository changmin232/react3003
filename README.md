# 리액트 포트폴리오

## 리액트 앱 만들기

npx create-react-app react3003

## 로우터 설치

npm install react-router-dom
yarn install react-router-dom

## axios 설치

npm install axios

## prop 설치

npm install prop-types

## scss 설치

npm install node-sass
npm install node-sass@4.14.1

## yarn 설치

npm install -g yarn

### 버전 확인

yarn -v;
yarn -version;
